# 🏫 AirSafe School

Air quality monitoring and analysis system for schools.

## Overview

This project builds a comprehensive air quality monitoring system that integrates:
- School registry data (DKI Jakarta)
- Weather data (Open-Meteo API)
- Azure Functions backend
- Coverage mapping and visualization

## Features

- Automated school registry downloads
- Weather data acquisition and processing
- Azure Functions for serverless data pipeline
- Coverage map visualization

## Tech Stack

- Python
- Azure Functions
- Open-Meteo API
